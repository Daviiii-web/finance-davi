PT 

Aplicativo de controle financeiro pessoal desenvolvido com React e TypeScript.

## Tecnologias

- React 18 + TypeScript
- Vite
- Recharts
- Tailwind CSS

## Funcionalidades

- Dashboard com saldo, receitas e despesas
- Gráficos de pizza, linha e barras
- Cadastro e exclusão de transações
- Filtro por tipo (receita/despesa)
- Maiores despesas por categoria



## Em desenvolvimento

- Banco de dados com Supabase
- Deploy na Vercel
- PWA para instalar no celular



**EN** | [PT](#financeflow)

Personal finance control app built with React and TypeScript.

## Tech Stack

- React 18 + TypeScript
- Vite
- Recharts
- Tailwind CSS

## Features

- Dashboard with balance, income and expenses
- Pie, line and bar charts
- Add and delete transactions
- Filter by type (income/expense)
- Top expenses by category


## In Progress

- Database with Supabase
- Deploy on Vercel
- PWA for mobile
